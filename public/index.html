URL Shortener Microservice

This is a url shortener microservice

Example usage:

Create new short url

    /new/http://www.example.com

Example response:

    {"shorturl": "https://url-shrt-zelite.herokuapp.com/4G",
    "longurl": "http://www.example.com"}

Follow short url

    https://url-shrt-zelite.herokuapp.com/4G",
